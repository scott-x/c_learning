# basic

### functions
- `printf`: printf, which prints to the screen, is defined in `stdio.h` header file.

### Variable

Variable can be defined like this :

```c
int foo;  //not initialized
int bar = 1;
```
and this

```c
int a = 0,b = 1,c = 2,d = 3, e = 4;
a = b - c + d * e;
printf("%d", a); /* will print 1-2+3*4 = 11 */
```
